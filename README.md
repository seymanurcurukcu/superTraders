SUPER TRADERS Install Project
-----------------------------

1. npm install
2. npx sequelize-cli db:create 
3. npm run migrate
4. npm run seed:all
5. npm run start:dev


Migration Remove
-------------------
** npm run migrate:undo
 
All Migration Remove
---------------------
** npm run migrate:undo:all

Seeders Remove
--------------
** npm run seed:undo:all
